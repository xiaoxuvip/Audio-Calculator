使用方法

三种方式，灵活选择

1

拖拽添加

直接将音频文件或包含音频的文件夹拖拽到软件窗口中，即可自动识别并统计。

2

右键菜单

点击软件右上角「添加右键」按钮（需管理员权限），之后在文件或文件夹上右键即可看到「音频时长统计」选项。

3

手动选择

点击底部「文件」或「文件夹」按钮，通过对话框选择要统计的音频。

⚡ 注意事项

  • Windows 系统限制：通过右键菜单选择超过 100 个文件时，系统只能传递部分文件。建议在文件夹上右键，软件会自动扫描文件夹内所有音频。
  • 管理员权限：添加/移除右键菜单需要管理员权限，请右键以管理员身份运行程序。
  • 图标刷新：添加右键菜单后，图标可能需要重启资源管理器或重新登录才能显示。
  • 首次运行：由于是单文件便携版，首次运行可能需要几秒钟解压运行时环境。
  • 文件大小说明：完整版 152MB 包含运行环境，开箱即用；轻量版约 2MB，需要安装 .NET 8 Desktop Runtime。
  • 轻量版运行时：如选择轻量版，请先安装 .NET 8 Desktop Runtime（选择 Windows x64 版本）。
